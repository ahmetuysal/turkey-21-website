Değişken Yaratma ve Kullanma

JavaScript içindeki değişkenler Java'da olduğu gibi sabit bir tipe sahip değildirler. Bu sebeple bir değişken yaratırken tipini belirtmek zorunda değilsiniz. Bunun yerine var ifadesini kullanacaksınız.

var a = 10;
var b = 20;
var c = a + b;

If İfadesi

If ifadeleri Java ile birebir aynıdır.

if (someBooleanFunction()) {
    return 1;
}

While Döngüleri

While döngüleri de Java ile aynıdır.

while (someCondition()) {
    sum += 1;
}

For Döngüleri

For döngüleri neredeyse aynıdır ancak sayaç değişkeni farklıdır.

// 0'dan 10'a (dahil) kadar olan sayıların toplamını bulur
var sum = 0;
for (var i = 0; i <= 10; i++) {
    sum += i;
}

Fonksiyonlar

private void ifadesini görmeye alıştığımız yerde bu kez function ifadesini göreceksiniz. Parametrelerin tiplerinin olmadığına da dikkat edin.

function someFunction (a, b) {
    return a === b;
}

Operatörler

Tüm operatörler aynıdır. (Karşılaştırma [==] operatörü hariç) Java^da karşılaştırma yapmak için iki eşittir işareti kullanılar ancak JavaScript için bu operatör === işareti ile gösterilir.

a === b; // a eşittir b
a !== b; // a eşit değildir b

Event Listeners

object.event = function(e) { 
    // your code  
}

Popüler Event Örnekleri

  • onclick
  • onmousedown
  • onmouseup
  • onkeypress
  • onkeydown
  • onkeyup
