Ortalama Metodu

Problemi yazan: Chris Piech ve Mehran Sahami

Bu egzersizde iki tane girdi alan ve bu girdilerin ortalamasını veren basit bir metot yazacağız:

private double average(double a, double b)

Çözüm

public class AverageMethod extends ConsoleProgram {

	public void run() {
		println(average(4, 5));
		println(average(10, 2));
		println(average(-4.1, 9.8));
	}

	private double average(double a, double b) {
		return (a + b) / 2;
	}
}
