Day4: MadMax.java

1. Mad Max

MadMax pragramını max metodunu doldurarak tamamlayın. Bu metot iki parametreden büyük olanını döndürür:

private int max(int i, int j)

Run metodu önceden yazılmıştır: 

public void run() {
    println("The max of 4 and 5 is: " + max(4, 5));
    println("The max of -4 and 4 is: " + max(-4, 4));
    println("The max of 1 and 2 is: " + max(1, 2));
    println("The max of 0 and 0 is: " + max(0, 0));
}

2. Ağaç çizme

Örnekler: İnsan Çizme, Yarısı Yeşil
Day4: DrawTrees.java

drawTree metodunu doldurun. Bu metot verilen (x,y) koordinatlarında tabanı ortalanmış bir ağaç çizer :

private void drawTree(int x, int y)

Run metodu önceden yazılmıştır: 

public void run() {
    drawTree(50, 150);
    drawTree(100, 300);
    drawTree(500, 200);
    drawTree(300, 250);
    drawTree(700, 310);
}

Her ağacın gövdesi 20 piksel genişliğinde ve 80 piksel yüksekliğinde olmalıdır. Yaprakların yarıçapları 30 piksel olmalıdır. Çıktınız aşağıdakine benzer olmalıdır:
